# Amazon-Sales-Analysis

Amazon Sales Analysis Dashboard | Power BI

This project presents an interactive Amazon Sales Analysis Dashboard built in Power BI, designed to visualize key sales metrics, product performance, and customer engagement trends. It provides a comprehensive overview of YTD (Year-to-Date) and QTD (Quarter-to-Date) sales, top-selling products, and category-wise insights to help drive data-driven decision-making.
